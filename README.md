# HemaLens - AI Blood Report Analyzer

HemaLens is a clinical-grade web application designed to help patients understand their laboratory data with clarity and precision. 

## Features
- **Biomarker Extraction**: Automatically identifies key parameters like RBC, Hemoglobin, Glucose, etc.
- **Bilingual Insights**: Provides summaries in both plain English and Hindi.
- **Actionable Advice**: Suggests dietary and lifestyle changes based on findings.
- **Risk Assessment**: Highlights critical values that require immediate attention.

## Tech Stack
- **HTML5**: Semantic structure for better accessibility and SEO.
- **Vanilla CSS**: Modern, responsive design with glassmorphism and bento-grid layout.
- **Vanilla Javascript**: Core logic for handling file uploads and data display.

## How to use
1. Open `index.html` in any modern browser.
2. Drag and drop your blood report image into the upload zone.
3. Wait for the AI to "see" and "interpret" the document (simulated for demo).
4. Review your personalized health dashboard.
